一部以純文字、瀏覽器狀態與不可靠病歷構成的心理恐怖網頁小說。

本作涉及家庭控制、性暴力的非露骨敘述、解離、自殺意念、自傷、精神醫療、藥物、排泄與身體恐怖。
建議年滿 18 歲且能自行評估心理狀態的讀者遊玩。
